# Google Colab Setup

Running notebooks on Colab provides free GPU access, which significantly reduces runtime for the three compute-heavy notebooks.

| Notebook | Local (CPU) | Colab (T4 GPU) |
|---|---|---|
| NB03 - Classic ML training | ~1-2 min | ~15 sec |
| NB04 - LSTM training | ~11 min | ~2-3 min |
| NB05 - DistilBERT inference (7,500 rows) | ~45 min | ~3-5 min |
| NB01, NB02 - EDA + Preprocessing | ~1-2 min | ~1-2 min |

---

## Prerequisites

Colab runs on Google's servers. The project repository can be cloned directly, but the dataset and trained model files are not in git and must be uploaded separately.

**Files not in git:**
- `data/IMDB Dataset.csv` (~63 MB)
- `data/imdb_preprocessed.csv` (~20 MB, generated by NB02)
- `models/` directory (~245 MB total, generated by NB03/NB04)

For a full pipeline run (NB01 - NB05 in order), only the raw dataset is needed upfront. Models are generated by the notebooks themselves.

## Step 3 - Upload Dataset and Models

### Option A: Google Drive (recommended - files persist between sessions)

Upload once and reuse across sessions:

1. Create `My Drive/imdb-project/` in Google Drive
2. Upload `data/IMDB Dataset.csv` into that folder
3. Upload all files from `models/` into `My Drive/imdb-project/models/`

```python
# Mount Drive and copy files into the cloned repo
from google.colab import drive
drive.mount('/content/drive')

import shutil, os
drive_base = '/content/drive/MyDrive/imdb-project'

os.makedirs('/content/imdb-sentiment-analysis/data', exist_ok=True)
shutil.copy(
    f'{drive_base}/IMDB Dataset.csv',
    '/content/imdb-sentiment-analysis/data/IMDB Dataset.csv'
)

# Only needed if running NB05 standalone (NB03/NB04 generate models when run)
if os.path.exists(f'{drive_base}/models'):
    shutil.copytree(
        f'{drive_base}/models',
        '/content/imdb-sentiment-analysis/models',
        dirs_exist_ok=True
    )
print('Ready.')
```

### Option B: Direct upload (no Drive required)

```python
from google.colab import files
import os

os.makedirs('/content/imdb-sentiment-analysis/data', exist_ok=True)
os.makedirs('/content/imdb-sentiment-analysis/models', exist_ok=True)

uploaded = files.upload()  # select IMDB Dataset.csv
for name, data in uploaded.items():
    with open(f'/content/imdb-sentiment-analysis/data/{name}', 'wb') as f:
        f.write(data)
```

## Step 4 - Running Notebooks

All five notebooks include automatic Colab path detection - no manual path changes are needed.

**Full pipeline (NB01 to NB05 in order):**

Open each notebook in Colab (File > Open notebook > Upload) and run all cells sequentially. The working directory is set to `notebooks/` by the `%cd` command in Step 2.

**NB05 standalone (model comparison only):**

NB05 loads pre-trained models from `models/`. Upload the model files via Step 3 before running.

**Sample sizes in NB05:**

The two constants in the NB05 setup cell control evaluation scope:

```python
COMMON_SAMPLE_SIZE = 7500   # rows for classic ML + LSTM evaluation
BERT_SAMPLE_SIZE   = 7500   # rows for DistilBERT evaluation
```

The GPU is auto-detected (`_DEVICE = 0` when CUDA is available).

---

## Step 5 - Save Results

Results (figures, CSVs, model files) are stored on Colab's VM and are lost when the session ends.

```python
# Zip and download all results
import shutil
from google.colab import files

shutil.make_archive('/content/results', 'zip',
                    '/content/imdb-sentiment-analysis/results')
files.download('/content/results.zip')
```

For model files generated by NB03/NB04:

```python
shutil.make_archive('/content/models', 'zip',
                    '/content/imdb-sentiment-analysis/models')
files.download('/content/models.zip')
```

Unzip locally and replace `results/` and `models/` directories accordingly.

- Free tier GPU allocation: ~4-12 hours per day. NB05 at `BERT_SAMPLE_SIZE=7500` takes ~15-20 minutes on T4.

- `imdb_preprocessed.csv` is generated by NB02. For an NB05-only run, upload it alongside the raw dataset (~20 MB).
