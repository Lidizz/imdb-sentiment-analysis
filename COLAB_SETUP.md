## How Colab works (important before starting)

Colab runs on Google's server, not your laptop. Each notebook you open in Colab gets its own isolated VM - if you open NB01 in one tab and NB02 in another, they are on completely separate machines and cannot share files.

**The correct approach for running all five notebooks in sequence:** open a single blank Colab notebook, run setup once, then execute all project notebooks from that one session using `nbconvert`. All notebooks share the same filesystem within the session, so NB03 can read what NB02 wrote, and so on.

Only the raw dataset needs to be uploaded. Models and preprocessed data are generated by the notebooks themselves.

## Step 1 - Open Colab and set GPU runtime

1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Click **File > New notebook** - this opens a blank notebook
3. Click **Runtime > Change runtime type**
4. Select **T4 GPU** under Hardware accelerator
5. Click **Save**

Do this before running any cells. Changing the runtime type restarts the kernel.

## Step 6 - Run all notebooks

Run this cell to execute all five notebooks in order. Each notebook saves its output files (preprocessed data, models, figures, CSVs) to disk, and the next notebook reads from them.

```python
import subprocess, sys

PROJECT = '/content/imdb-sentiment-analysis'

# Per-notebook timeouts (seconds). NB02 lemmatizes 35K reviews; NB04 trains LSTM.
notebooks = [
    ('notebooks/01_data_exploration.ipynb',    300),
    ('notebooks/02_preprocessing.ipynb',      1800),
    ('notebooks/03_classic_ml_models.ipynb',   600),
    ('notebooks/04_deep_learning_model.ipynb', 7200),
    ('notebooks/05_model_comparison.ipynb',    3600),
]

for nb, timeout in notebooks:
    print(f'\nRunning {nb} ...')
    result = subprocess.run(
        [
            sys.executable, '-m', 'jupyter', 'nbconvert',
            '--to', 'notebook',
            '--execute',
            '--inplace',
            f'--ExecutePreprocessor.timeout={timeout}',
            nb,
        ],
        capture_output=True, text=True,
        cwd=PROJECT,
    )
    if result.returncode == 0:
        print(f'  Done: {nb}')
    else:
        print(f'  FAILED: {nb}')
        print(result.stderr[-2000:])
        break
```

Expected total runtime on T4 GPU: **~25-35 minutes** for all five notebooks.

## Step 7 - Download results
